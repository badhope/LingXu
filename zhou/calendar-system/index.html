实用换算

公历年份 → 干支年

公式: (公历年份 - 3) % 60

结果为0-59，对应六十甲子序号

2024年: (2024-3)%60=37 → 戊子

干支年 → 公历年份

公式: 60×n + 甲子年 + 3

甲子年为公历4年或往前推算

甲子年: 60×n + 4 + 3

月干支计算

年干 × 5 + 月支 = 月干序号

简化: 甲己年起甲寅，乙庚年起丙寅...

甲年寅月为丙寅月

修行与时间

闭关计时

闭关通常以"须臾"或"刻"计时。一须臾约48分钟，一刻约2小时。

灵气潮汐

灵气随时间有潮汐变化。子时、午时为灵气交汇之机，修行者当把握。

劫运周期

天地大劫以"元"为单位，一元为60年。末法时代，劫运连连，修行者需谨慎。
