# DATA-AI

DATA-AI - 智能化数据处理和分析平台，支持多种 AI 工具和代理

## 功能特性

- 🤖 多功能 AI 代理
- 📊 数据分析和可视化
- 🌐 网页浏览和自动化
- 🔍 智能搜索工具
- 📦 MCP 协议支持
- 🎯 企业级界面设计

## 安装

### 使用 pip

```bash
pip install -r requirements.txt
```

### 配置

复制配置文件并编辑：

```bash
cp config/config.example.toml config/config.toml
```

## 快速开始

```bash
python main.py
```

## 项目结构

```
DATA-AI/
├── app/              # 应用核心代码
├── config/           # 配置文件
├── assets/           # 资源文件
└── tests/            # 测试代码
```
