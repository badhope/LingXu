## 🎯 项目介绍

**灵墟档案馆** 是一个沉浸式探索中华玄学与失落修行文明的数字档案馆。

> *公元2026年，末法时代，灵气凋零已过千年。历史被封印，人们已经遗忘了那些真实存在过的修行者——老子飞升、庄子化蝶...这些不是神话，而是曾经真实存在过的修行文明。*

> **灵墟档案馆** 致力于保存这些被遗忘的修行文明，探索天地人神鬼的终极奥秘。

## ⚠️ 开发注意事项

### 🚨 CSS Modules 致命陷阱

> ⚠️ **绝对不要在 `design-tokens.scss` 中添加 `:root` 选择器！**
>
> 会导致全站 CSS Modules 崩溃，所有页面报 500 错误。
>
> ✅ **正确位置**：添加全局变量请去 `xianxia-css3.scss`

### 📝 错误速查表

| 错误信息 | 10秒解决方案 |
|---------|-------------|
| `Selector ":root" is not pure` | 检查 design-tokens.scss 头部 |
| 模板字符串不渲染变量 | 是不是少写了 `$`？ |
| 导航滚动穿透 | `overscroll-behavior: contain` |
