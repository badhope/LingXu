## 🔧 开发指南

### 快速开始

1. **启动服务**
```bash
cd /workspace/OpenManus
python web_app.py
```

2. **访问界面**
打开浏览器访问：`http://localhost:8000/`

3. **配置API Key**
- 点击侧边栏"⚙️ 设置"
- 选择模型提供商
- 输入API Key
- 保存设置

4. **开始使用**
- 在主界面输入您的需求
- AI将自动识别意图并执行相应操作
- 支持代码生成、图表绘制、数据分析等

### API文档

#### WebSocket接口
- **端点**：`ws://localhost:8000/ws`
- **认证**：通过设置中的API Key
- **消息格式**：JSON

#### REST API
- `GET /api/settings` - 获取设置
- `POST /api/settings` - 保存设置
- `GET/POST /api/knowledge-bases` - 知识库管理
- `GET/POST /api/skills` - 技能管理
- `GET/POST /api/mcp/servers` - MCP服务器管理

### 扩展开发

#### 添加新的MCP服务器
```python
# 在设置中添加MCP服务器
{
    "name": "我的服务器",
    "type": "process",  // 或 "http"
    "command": "npx @modelcontextprotocol/server-filesystem",
    "enabled": true
}
```

#### 创建自定义技能
```javascript
// 在技能系统中创建
{
    "name": "我的技能",
    "icon": "⚡",
    "description": "技能描述",
    "parameters": [],
    "prompts": {}
}
```

## 🐛 故障排除

### 问题1：文件上传失败
**可能原因**：
- 文件格式不支持
- 文件大小超过50MB
- 网络连接问题

**解决方案**：
- 检查文件格式是否在支持列表中
- 压缩或分割大文件
- 检查网络连接

### 问题2：API调用失败
**可能原因**：
- API Key无效或过期
- 配额用尽
- 网络问题

**解决方案**：
- 检查API Key是否正确
- 查看API服务商的控制台
- 重试请求

### 问题3：移动端显示异常
**可能原因**：
- 浏览器缓存问题
- 屏幕尺寸不兼容

**解决方案**：
- 清除浏览器缓存
- 强制刷新页面（Ctrl+Shift+R）
- 尝试其他浏览器
