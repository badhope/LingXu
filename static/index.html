❓ 使用说明

🚀 快速开始

DATA-AI 是一个完整的智能助手系统，包含以下核心功能：

  • 万能对话 - 支持自然语言对话，自动识别意图
  • 代码执行 - 自动生成并执行 Python 代码
  • 图表生成 - 支持 matplotlib 等可视化库
  • 知识库管理 - 上传文档，智能检索
  • 技能系统 - 自定义提示词和技能
  • MCP工具 - 支持 Model Context Protocol

💡 使用示例

生成一个正弦函数的折线图
分析这些数据：[1,2,3,4,5]
帮我写一个斐波那契数列
解释这个Python代码

📚 知识库说明

1. 创建知识库 → 2. 上传文档 → 3. 配置清洗规则 → 4. 提问时自动检索

⚡ 技能系统

技能是预定义的提示词模板，可以自定义参数和功能。参考规范：config/schema/skill_schema.json

🔌 MCP协议

MCP (Model Context Protocol) 允许连接外部工具和数据源。配置文件：config/mcp.json

📋 规范文档

所有系统规范定义在 config/schema/ 目录下：

  • knowledge_base_schema.json - 知识库系统规范
  • skill_schema.json - 技能系统规范
  • mcp_schema.json - MCP工具规范
