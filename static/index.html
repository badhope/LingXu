使用说明

快速开始

DATA-AI 是一个完整的智能助手系统，包含以下核心功能：

  • 万能对话 - 支持自然语言对话，自动识别意图
  • 代码执行 - 自动生成并执行 Python 代码
  • 图表生成 - 支持 matplotlib 等可视化库
  • 知识库管理 - 上传文档，智能检索
  • 技能系统 - 自定义提示词和技能
  • MCP工具 - 支持 Model Context Protocol

快捷键

  • Enter - 发送消息
  • Shift + Enter - 换行
  • Ctrl/Cmd + N - 新建对话
  • Ctrl/Cmd + / - 快捷键帮助
  • Escape - 关闭面板 / 停止生成
  • Up Arrow - 编辑最后一条用户消息
