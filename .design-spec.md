# 天机 - 项目设计规范文档

## 1. 设计风格

### 核心美学：暗金神秘风
- **整体气质**：深邃神秘、庄重典雅、玄学氛围浓厚
- **视觉关键词**：暗夜、金色流光、八卦玄学、东方神秘主义
- **设计语言**：深色背景 + 金色点缀 + 微光粒子 + 流动线条

### 关键特征
- **背景层次**：深色渐变底色 + 动态粒子 + 光晕效果
- **视觉焦点**：金色作为唯一高亮色，引导用户注意力
- **动画风格**：缓慢、优雅、有仪式感的过渡动画
- **纹理质感**：金属质感、丝绸光泽、古卷纹理

## 3. 字体排版

### 字体家族

#### 标题字体
- **主标题**：`Ma Shan Zheng` (马善政楷书) - 传统书法感
- **副标题**：`Noto Serif SC` (思源宋体) - 典雅宋体
- **装饰字**：`ZCOOL XiaoWei` (站酷小薇体) - 特殊装饰

#### 正文字体
- **正文**：`Noto Sans SC` (思源黑体) - 清晰易读
- **数字**：`Cinzel` - 西方古典数字，用于日期
- **代码/特殊**：`JetBrains Mono` - 等宽字体

### 字体层级
```css
/* 标题层级 */
--font-hero: 72px / 1.2 'Ma Shan Zheng', cursive;      /* 首屏大标题 */
--font-h1: 48px / 1.3 'Ma Shan Zheng', cursive;        /* 一级标题 */
--font-h2: 36px / 1.4 'Noto Serif SC', serif;          /* 二级标题 */
--font-h3: 28px / 1.4 'Noto Serif SC', serif;          /* 三级标题 */
--font-h4: 22px / 1.5 'Noto Sans SC', sans-serif;      /* 四级标题 */

/* 正文层级 */
--font-body-lg: 18px / 1.6 'Noto Sans SC', sans-serif; /* 大正文 */
--font-body: 16px / 1.6 'Noto Sans SC', sans-serif;    /* 标准正文 */
--font-body-sm: 14px / 1.5 'Noto Sans SC', sans-serif; /* 小正文 */
--font-caption: 12px / 1.4 'Noto Sans SC', sans-serif; /* 说明文字 */

/* 特殊 */
--font-number: 24px / 1 'Cinzel', serif;               /* 数字展示 */
--font-date: 64px / 1 'Cinzel', serif;                 /* 日期大数字 */
```

### 字重规范
- **标题**：400 (Regular) - 书法字体不需要加粗
- **正文**：400 (Regular)
- **强调**：500 (Medium)
- **辅助**：300 (Light)

## 5. Logo设计

### Logo概念
- **主体**：八卦图 + "天机"书法字
- **风格**：金色描边 + 深色填充
- **动效**：加载时八卦缓慢旋转

### Logo变体
| 变体 | 尺寸 | 使用场景 |
|------|------|----------|
| 完整版 | 180x60px | 首页顶部 |
| 图标版 | 48x48px | favicon、小尺寸 |

## 7. 动画规范

### 过渡时间
| 类型 | 时长 | 缓动函数 |
|------|------|----------|
| 快速交互 | 150ms | ease-out |
| 标准过渡 | 300ms | ease-in-out |
| 页面切换 | 500ms | cubic-bezier(0.4, 0, 0.2, 1) |
| 仪式动画 | 3000-5000ms | cubic-bezier(0.25, 0.1, 0.25, 1) |

### 关键动画
```css
/* 八卦旋转 */
@keyframes bagua-rotate {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}

/* 金光闪烁 */
@keyframes gold-shimmer {
  0%, 100% { opacity: 0.5; }
  50% { opacity: 1; }
}

/* 淡入上移 */
@keyframes fade-up {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

/* 粒子漂浮 */
@keyframes particle-float {
  0%, 100% {
    transform: translateY(0) translateX(0);
    opacity: 0.3;
  }
  50% {
    transform: translateY(-20px) translateX(10px);
    opacity: 0.8;
  }
}
```

## 9. 无障碍规范

- **对比度**：文字与背景对比度 ≥ 4.5:1
- **焦点状态**：所有交互元素需有明显的焦点样式
- **动画控制**：提供动画开关，尊重 prefers-reduced-motion
- **语义化**：正确使用 HTML5 语义标签
- **键盘导航**：所有功能可通过键盘访问
