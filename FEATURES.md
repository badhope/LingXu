## 📋 技术实现细节

### 文件结构
```
/workspace/OpenManus/
├── web_app.py                 # 主应用文件
├── config/                    # 配置目录
│   └── schema/               # Schema规范
├── data/                     # 数据目录（自动创建）
│   ├── knowledge_bases/      # 知识库存储
│   ├── skills/               # 技能存储
│   └── uploads/              # 文件上传存储
├── CHANGELOG.md              # 更新说明
└── SYSTEM_GUIDE.md           # 系统指南
```

### 核心API端点
- `POST /api/knowledge-bases/{kb_id}/documents` - 文件上传
- `POST /api/skills/generate` - AI技能生成
- `GET/POST /api/settings` - 设置管理
- `GET/POST /api/knowledge-bases` - 知识库管理
- `GET/POST /api/skills` - 技能管理
- `GET/POST /api/mcp/servers` - MCP服务器管理

## 🧪 使用指南

### 快速开始
1. 启动服务：`python web_app.py`
2. 访问界面：`http://localhost:8000`
3. 配置API Key：在设置中配置阿里云/OpenAI等模型

### 文件上传
1. 进入知识库页面
2. 点击"上传文档"区域
3. 选择一个或多个文件
4. 等待上传完成

### AI技能生成
1. 进入技能系统页面
2. 点击"创建技能"标签
3. 填写技能用途（如："Python代码审查"）
4. 点击"AI辅助生成"
5. 自动生成后点击"创建技能"

### MCP服务器配置
1. 进入MCP工具页面
2. 点击任一快速添加按钮（如"📁 文件系统"）
3. 补充必要的环境变量（根据需要）
4. 点击"添加服务器"
